Position

The Council logo must always appear on branded websites in the position set in the template (on the right in the masthead). The vertical line of the logo should extend to the top of the screen.

For partner websites, or commercial Council services that have their own branding (such as Oxford Direct Services or Oxford Town Hall) the logo is placed in the footer on the right hand side.

Logo file

The template file contains an .svg and a .png version of the logo for use. The .svg file is preferred.

The custom.css file contains an amended link to the .png file (commented out) should it be required instead.
